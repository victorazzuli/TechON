# 🚀 Tech On - E-commerce de Tecnologia


A **Tech On** é uma plataforma de e-commerce moderna e responsiva focada na venda de acessórios premium (áudio e energia). O projeto simula uma experiência real de compra, desde a navegação por categorias até o checkout dinâmico.

## 💻 Sobre o Projeto

Este site foi desenvolvido para aplicar conceitos avançados de manipulação de DOM e persistência de dados no navegador. Ele conta com um sistema dinâmico onde os detalhes dos produtos são carregados via JavaScript, evitando a redundância de código e simulando o comportamento de aplicações profissionais.

## ✨ Funcionalidades

- **Catálogo Dinâmico:** Divisão entre produtos originais e linha premium com cards interativos.
- **Página "Saiba Mais":** Carregamento dinâmico de descrições e especificações técnicas via parâmetros de URL.
- **Carrinho de Compras:**
  - Adição de itens com verificação de duplicidade.
  - Controle de quantidade e remoção de itens.
  - Cálculo automático de subtotal e frete.
- **Persistência com LocalStorage:** O carrinho de compras não é perdido ao atualizar ou fechar a página.
- **Checkout Simulado:** Finalização de compra com geração de QR Code dinâmico para pagamento via Pix.
- **Interface Responsiva:** Totalmente adaptado para tablets e smartphones.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Design moderno utilizando Flexbox e CSS Grid.
- **JavaScript (ES6+):** Lógica do carrinho, manipulação de dados e integração de API de QR Code.
- **Google Fonts:** Tipografia focada em UX (Poppins).

## 📁 Estrutura de Arquivos

- `Techon.html`: Landing Page principal.
- `produtostech.html`: Vitrine de produtos.
- `saiba.html`: Página dinâmica de detalhes do item.
- `carrinhotech.html`: Gerenciamento de itens e sistema de pagamento.
- `sobretech.html` / `contatotech.html`: Páginas institucionais.
- `estilo4.css`: Folha de estilo centralizada.
